Setup "ohne Flugminutenkonto":

in global.conf:
---------------
unter <admin>..:
password ändern

unter <jetty>..:
port auf einen freien Port einstellen, auf diesem ist eine rudimentäre Web-Verwaltungsoberfläche erreichbar 

in default.conf:
----------------

unter <server>..:
il2port anpassen (Konsolenport vom Il2-Server siehe confs.ini: [Console]\nIP=Portnummer)
consoleport anpassen (einen freien Port auswählen, der dann für weitere Konsolenclients zur Verfügung steht)

unter <fbdj>..:
installationPath auf eine FBDj-Installation mit Version 1.5b verweisen, ggf. / statt \ verwenden

unter <fbdj><DCG>..:
minutesPerMission anpassen

unter <game> ist homeAlone=true eingestellt, dadurch werden sämtliche experimentellen Sturmonanny Funktionen 
in Sachen Minutenkonto, Deathkick-ersatz (deathpause) etc. deaktiviert, relevant sind dann nur die Einstellungen 
unter <server> und unter <fbdj>.
